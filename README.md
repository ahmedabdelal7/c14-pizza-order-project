## 📌 Description
A Windows Forms application built with C# that allows users to customize their pizza by selecting size, toppings, crust type, and dining option.

The project focuses on applying and connecting different UI controls while dynamically updating the order summary and total price based on user interactions.

---

## ⚙️ Features
- Select pizza size (Small, Medium, Large)
- Choose multiple toppings
- Select crust type (Thin / Thick)
- Choose dining option (Eat In / Take Away)
- Real-time price calculation
- Dynamic order summary update
- Reset form to default values

---

## 🧠 Concepts Used
- Windows Forms (WinForms)
- Event Handling
- Working with UI Controls (RadioButton, CheckBox, GroupBox)
- Methods organization
- Conditional logic

---

## 🏗️ How the System Works
1. The user selects pizza size, toppings, crust type, and where to eat.
2. Each selection triggers an event.
3. The system calculates:
   - Size price
   - Toppings price
   - Crust price
4. The total price is calculated using a dedicated method.
5. The order summary is updated in real-time.
6. The user can reset all selections using the reset button.
